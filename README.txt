ANDRO PARK

Andro Park an android application used for easy searching for parking spots, knowing the availability of the parking spots.
Here we have used the dynamic MySql database.

 
Here we have attached the following files and folders:

1.Andro Park.pdf
	
	It has the idea and the basic concept of our project.

2.Ardunio
	
	This folder has the codes used to connect the RFID sensor to MySql using Ardunio UNO.
	It also contains photos of the hardware and the COM screen.

3.Database

	It contains the database file(.sql) of our android application.
	It also has the photo of the design of the database.

4.temp_map2
	
	It is the android studio project of our application which has all the source code of our application.
